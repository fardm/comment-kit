من از این پروژه برای سایتم که با quartz ساخته شده استفاده کردم.
خلاصه تغییراتی که دادم اینه:

- فارسی کردن متن ها
- تاریخ شمسی
- بهینه کردن استایل
- اضافه کردن دارک مود به پنل ادمین
- باز طراحی ری‌اکشن با ایموجی (قبلا شبیه Disqus بود من شبیه گیت هاب کردمش که بشه ایموجی دلخواه وارد کرد)
