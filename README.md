این پروژه برای اضافه کردن قابلیت کامنت به سایت های استاتیک هست. من برای سایتم که با کوارتز ساخته شده از این ابزار استفاده کردم. 
ریپوی اصلی [اینجاست](https://github.com/dlnorman/standalone-comments). برای استفاده از کوارتز کمی کدش رو تغییر دادم.

خلاصه تغییراتی که انجام دادم اینه:


- فارسی کردن متن ها
- نمایش تاریخ ها به صورت شمسی
- بهینه کردن استایل
- اضافه کردن دارک مود به پنل ادمین
- باز طراحی ری‌اکشن با ایموجی (قبلا شبیه Disqus بود که فقط 4 تا ایموجی داشت. الان شبیه ری اکشن گیت هاب شده ایموجی های بیشتری داره.)


آموزش استفاده ازش رو [اینجا](https://ifard.ir/self-hosted-quartz#%DA%A9%D8%A7%D9%85%D9%86%D8%AA) توضیح دادم.
