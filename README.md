این ابزار برای اضافه کردن کامنت به سایت های استاتیکه. من از این پروژه برای سایت خودم که با کوراتز ساخته شده استفاده کردم. 


یه مقدار هم مجبور شدم کدش رو تغییر بدم. خلاصه تغییرات اینه:

- فارسی کردن متن ها
- نمایش تاریخ ها به صورت شمسی شمسی
- بهینه کردن استایل
- اضافه کردن دارک مود به پنل ادمین
- باز طراحی ری‌اکشن با ایموجی (قبلا شبیه Disqus بود من شبیه گیت هاب کردمش که بشه ایموجی دلخواه وارد کرد)
