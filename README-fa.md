# کامنت مستقل

این پروژه برای اضافه کردن قابلیت کامنت به [Quartz](https://quartz.jzhao.xyz/) هست. ریپوی اصلی [اینجاست](https://github.com/dlnorman/standalone-comments). من کمی کدش رو تغییر دادم تا برای کوارتز بهینه باشه.

## امکانات
- ثبت ری اکشن با 10 تا ایموجی هم برای یادداشت هم برای کامنت
- پنل ادمین برای مشاهده و مدیریت کامنت ها
- قابلیت ایمپورت و اکسپورت داده‌ها
- تنظیم ایمیل برای دریافت نوتیفیکیشن کامنت جدید

<br>

## نصب
ترجیحا از PHP ورژن 8.0 به بعد استفاده کنید. سازندش نوشته بود از ورژن 7.4 پشتیبانی میکنه ولی من با 8.1 هم ارور گرفتم. وقتی گذاشتم روی 8.3 برطرف شد.
### گام اول: نصب روی سرور

1. یه هاست اشتراکی تهیه کنید و یک ساب دامین بسازید. مثلا `comments.yourdomain.com`.
2. همین ریپازتوری رو دانلود کنید. روی دکمه سبز رنگ Code کلیک کنید، گزینه Download ZIP رو انتخاب کنید.
3. فایل zip رو روی هاست‌تون توی پوشه `public html` آپلود کنید و Extract کنید.
4. فایل `config.php` رو ادیت کنید:
	- بخش `APP_URL`  آدرس همین ساب دامینی که فایل ها رو آپلود کردید وارد کنید: `comments.yourdomain.com`
	- بخش `ALLOWED_ORIGINS` آدرس سایت تون(کوارتز) رو وارد کنید: `yourdomain.com`
5. فایل `set-password.php` رو با رفتن به این آدرس باز کنید: `https://comments.yourdomain.com/set-password.php`
6. یک پسورد انتخاب کنید.
7. پنل ادمین رو با رفتن به این آدرس باز کنید: `https://comments.yourdomain.com/admin.html`
8. پسوردی که انتخاب کرده بودید رو تایپ کنید و وارد پنل بشید.

بعد از اینکه وارد شدید فایل `set-password.php` رو از پوشه `public html` حذف کنید.

بعد از اینکه وارد شدید فایل ست پسورد رو حذف کنید.

### گام دوم: نصب روی کوارتز
توی ترمینال کوارتز این دستور رو وارد کنید تا پلاگین این کامت روی کوارتز نصب بشه:
```
npx quartz plugin add github:fardm/quartz-standalone-comments
```

وقتی که نصب شد فایل `quartz.config.yaml` رو باز کنید. توی تنظیمات پلاگین فیلد `backendUrl` مسیر سرور خودتون رو وارد کنید. یعنی همین ساب دامینی که فایل های کامنت رو آپلود کردید. اینطوری:

```yaml
- source: github:fardm/quartz-standalone-comments
enabled: true
options:
  backendUrl: https://comments.yourdomain.com
layout:
  position: afterBody
  priority: 100
```

دستور زیر رو بزنید تا پیش نمایش سایت رو ببنید:
```
npx quartz build --serve
```
حالا باید کامنت به انتهای صفحه اضافه شده باشه.

البته الان فقط پیش نمایشه و چون لوکال هست ری اکشن ها و کامنت ها توی دیتابیس ذخیره نمیشه. دستور زیر رو بزنید تا سایت تون آپدیت و سینک بشه:
```
npx quartz sync
```

سایت رو چک کنید. ری کشن و کامنت رو تست کنید.

## فعال کردن نوتیفیکشن ایمیل
1. به پنل هاست تون برید و گزینه Cronjobs رو انتخاب کنید.
2. توی بخش کامند این دستور رو وارد کنید:
```
php /home/username/public_html/comments/utils/process-email-queue.php
```
3. به پنل ادمین کامنت برید. توی بخش utilities گزینه Email Notifications رو فعال کنید.
4. توی فیلد Admin Email ایمیل تون رو وارد کنید و ذخیره کنید.
5. 
چند تا کامنت تستی وارد کنید ببینید کار میکنه یا نه. توی پنل ادمین هم از بخش Test Email میتونید یک ایمیل تستی بفرستید.


## تغییرات من
خلاصه تغییراتی که به این پروژه اضافه کردم:
- فارسی کردن متن ها
- نمایش تاریخ ها به صورت شمسی
- بهینه کردن استایل
- اضافه کردن دارک مود به پنل ادمین
- باز طراحی ری‌اکشن با ایموجی (قبلا شبیه Disqus بود که فقط 4 تا ایموجی داشت. الان شبیه ری اکشن گیت هاب شده ایموجی های بیشتری داره.)
- اضافه کردن راهنمای فرمت دهی در باکس متن کامنت

<br>

## امنیت
برای کاهش اسپم و سوءاستفاده، سیستم از چند لایه محافظتی استفاده می‌کند:
- استفاده از Honeypot برای شناسایی و مسدود کردن ربات‌ها
- محدودیت نرخ درخواست‌ها (Rate Limiting)
- مسدود شدن IP پس از چند بار وارد کردن رمز عبور اشتباه در پنل مدیریت
